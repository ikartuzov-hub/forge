## The problem
Most ideas die in the gap between "I have an idea" and a thing that actually runs. The distance from a single sentence to a living asset is where momentum is lost — not for lack of ideas, but for lack of a path.

## The build
Forge is the SeedWave constructor: describe an idea in one phrase, follow the path of the idea, and walk out with a working MVP. It is the method made visible — idea → MVP → asset — and the clearest statement of what this studio does.

## Stack
`SeedWave engine` · `multilingual UI` · `GitHub Pages` · `OpenAI / Claude` · `Notion + Make (intake pipeline)`

- **The idea → MVP path itself** — a repeatable intake that turns a one-line concept into a scoped, buildable product, not a brainstorm.
